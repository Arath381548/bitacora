Inf. 
        HTML5 es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web

CSS y capas

Durante el periodo de esta clase se adquirio conocimiento del que es el css y las capas que tiene una pagina web, asi como se interactua en las capas.

Inf. 
        CSS(Cascading Style Sheets)es un lenguaje que maneja el diseño y presentación de las páginas web, 
        es decir, cómo lucen cuando un usuario las visita.Funciona junto con el lenguaje HTML que se encarga 
        del contenido básico de las páginas.

Hoja de estilo

Durante el periodo de esta clase se dio informacion acerca de las hoja de estilos, asi como una breve explicacion de como y donde consegir estilos en sitios web.

Inf. 
        Una hoja de estilo es un archivo de extensión *.CSS (CSS, Cascading Style Sheets = Hojas de estilo)
        que contempla definiciones de formato (tipo de fuente, tamaño, color de la fuente, color de fondo,
        párrafos, etc) de las distintas etiquetas que forman una página *.HTML.
